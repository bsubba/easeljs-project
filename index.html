    Interactive Canvas

    This application is created as a part of coding test. An user can double click within the canvas area with light green background. A point denoted with a small red circle is created upon double click. User can select three arbitrary points within the canvas, which will eventually create a blue parallelogram and a yellow circle with same area and centroid based on those three selected points. Location information of the all the diagrams created are displayed in terms of (x,y) co-ordinates. User is free to move around all the circles created. The changed location of the diagrams are displayed upon moving around.

    This is a simple demonstration of how html5 canvas element can be used to create interactive application to create different types of diagrams. To make the work more easier and swift, a javascript library, “easeljs” has been used for this project.
